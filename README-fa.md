# مدیریت مالی

یک برنامه جامع مدیریت مالی شخصی با نسخه‌های فارسی و انگلیسی

## نسخه‌های زبانی

### 🇮🇷 نسخه فارسی
- **شاخه**: `persian`
- **تقویم**: شمسی
- **واحد پول**: تومان
- [دانلود آخرین نسخه فارسی](https://github.com/your-username/financial-manager/releases/latest?q=persian&expanded=true)

### 🇬🇧 نسخه انگلیسی  
- **شاخه**: `main`
- **تقویم**: میلادی
- **واحد پول**: دلار
- [Download Latest English Version](https://github.com/your-username/financial-manager/releases/latest)

## 📥 نصب

### فایل اجرایی ویندوز (پیشنهادی)
فایل `.exe` مناسب را از صفحه [Releases](https://github.com/your-username/financial-manager/releases) دانلود کنید.

### از طریق کد منبع
```bash
# نسخه فارسی
git clone -b persian https://github.com/your-username/financial-manager.git
cd financial-manager
pip install -r requirements.txt
python main_fa.py

# نسخه انگلیسی
git clone -b main https://github.com/your-username/financial-manager.git
cd financial-manager
pip install -r requirements.txt
python main_en.py
